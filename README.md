# AI Job-CV Matcher

An AI-powered application that compares a resume with a job description, identifies skill gaps, and provides practical improvement suggestions using NLP and LLM techniques.

## Project Goal

The goal of this project is to build a practical AI assistant for job applications. The system will help users understand how well their CV matches a job description and what they can improve before applying.

## Planned Features

- Compare CV text with job description text
- Calculate a basic match score
- Identify matching skills
- Identify missing or weak skills
- Generate short improvement suggestions
- Optional: create interview preparation questions
- Optional: generate a tailored cover letter draft

## Tech Stack

Planned:

- Python
- Streamlit
- scikit-learn
- Sentence Transformers
- Hugging Face
- LLM API or open-source LLM
- GitHub documentation
